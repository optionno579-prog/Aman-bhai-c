# Telegram Bot Setup Instructions

## Prerequisites
- Ensure you have Python 3.7 or higher installed.
- Install pip for managing Python packages.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/optionno579-prog/Aman-bhai-c.git
   cd Aman-bhai-c
   ```
2. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Configuration
1. Create a new bot using BotFather on Telegram and get your bot token.
2. Rename the `.env.example` file to `.env` and add your bot token:
   ```
   TELEGRAM_BOT_TOKEN=your_token_here
   ```

## Running the Bot
Execute the following command to start the bot:
```bash
python bot.py
```

## Usage
- Send commands to your bot in Telegram to interact with it.
